### 1. はじめに

#### NOCの目的と概要
NOC (Network Operation Center) は各種カンファレンスでボランティアとしてWi-Fiを提供しています。

#### ドキュメントの目的と構成
各種イベントでWi-Fiを提供するためのノウハウをある程度まとめておいて、いろんなイベントで使い回せるようにするためのドキュメントです。
